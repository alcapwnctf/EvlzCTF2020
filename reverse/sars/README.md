# protobuf-forward-compat

Protocol buffer forward compatibility challenge.

## Build

```bash
    make
```
(protoc, protoc-gen-go required in path)

## Usage

```bash
    ./build/server 0.0.0.0:8000
```

```bash
    ./build/client -host 127.0.0.1 -port 7800
```
